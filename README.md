## Project Overview

Developed a **comprehensive, interactive, and production-ready Power BI dashboard** for **Blinkit** to analyze sales performance, outlet efficiency, product trends, and customer satisfaction across India.

This dashboard empowers stakeholders with real-time insights to drive faster decision-making in the fast-paced 10-minute delivery ecosystem.

### Key Achievements
- Built **end-to-end** from scratch in just a few weeks
- Analyzed **1.20 Million records**
- Designed dynamic KPIs and advanced DAX measures
- Delivered professional UI/UX with seamless interactivity
- Identified high-performing outlets & products + root cause analysis for underperformance and wastage

---

## Key Features

| Feature                        | Description |
|-------------------------------|-----------|
| **Executive KPI Cards**       | Total Sales, AOV, Quantity Sold, Avg Customer Rating with MoM trends |
| **Outlet Performance**        | Analysis by Type, Size & City Tier (Tier 1/2/3) |
| **Product Insights**          | Deep dive into Dairy, Fruits & Vegetables, Beverages, Snacks, etc. |
| **Dynamic Matrix**            | Switch between Sales, Quantity, AOV & Rating using slicers |
| **Advanced Interactivity**    | Cross-filtering, Drill-through, Bookmarks, Reset All Filters |
| **Professional Design**       | Clean UI/UX with mobile responsiveness |

## Project Structure

**1. Business Objective**  
Analyze Blinkit’s sales data to optimize revenue, reduce wastage, and improve outlet & product performance.

**2. Data Processing**  
- ETL using Power Query  
- Data cleaning, transformation & quality checks  
- Star Schema modeling with proper relationships

**3. Dashboard Development**  
- Executive KPIs  
- Outlet, Product Category & Location Tier Analysis  
- Dynamic & Interactive Visualizations

**4. Insights & Recommendations**  
- Dynamic discounting strategies  
- Inventory optimization to reduce stock wastage  
- Actionable recommendations for underperforming outlets
